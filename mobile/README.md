# Mobile app (Expo)

This is a minimal Expo shell to bootstrap the mobile app.

## Install

```bash
cd mobile
npm install
```

## Run

```bash
npm run start
```

## Environment

Set the backend URL in your shell before running Expo:

```bash
export EXPO_PUBLIC_API_BASE="http://127.0.0.1:8000"
```

Then use it in your mobile API client.
